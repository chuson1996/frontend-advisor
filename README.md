# FrontendAdvisor

Run local mongo database
`mongod`

To drop database
```
use codeadvisor
db.dropDatabase()
```

Install babel-cli by
```
npm install -g babel-cli
```

To populate data
```
babel-node mongoose-test.js
```

Rules to tag resources:
1. Lowercase
2. Seperate words by spaces
